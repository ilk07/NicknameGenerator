# Многопоточное программирование

## Переменные многопоточной программы

### Генерация никнеймов
  
Считаем красивые ники на 100_000 вариантов из "abc":
- палиндромы;
- из одинаковых букв;
- из букв в порядке возрастания (английский алфавит).
